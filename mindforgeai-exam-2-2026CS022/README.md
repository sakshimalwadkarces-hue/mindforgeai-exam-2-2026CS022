# MindforgeAI Exam 2 Dashboard

This repository now includes a visual landing page at [index.html](index.html) that helps you navigate the full exam structure.

## What is included
- Section A to F navigation cards
- Direct links to solutions, notes, datasets, notebooks, dashboards, and screenshots
- A simple GitHub Pages-friendly structure for easy sharing

## Deploy to GitHub Pages
1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings -> Pages.
4. Choose the main branch and root folder.
5. Save and wait for the site to publish.

Your dashboard will then be available at:
https://<your-username>.github.io/<your-repository-name>/
