II. Component Responsibilities

💻

VS Code

Role: Primary IDE.
Manages the project directory, edits documentation, and utilizes the terminal to push the finalized codebase to Git version control.

📓

Local Jupyter

Role: Offline Python space.
Facilitates the authoring, iterative testing, and logical verification of the data analytics pipeline against the local dataset.

☁️

Google Colab

Role: Cloud validation.
Ensures the executed notebook runs flawlessly in an isolated environment without local dependencies, generating a shareable URL.

🐙

GitHub Repository

Role: Version control.
Securely stores the comprehensive "Engineering Package" (code, data, README, and terminal screenshots) for review.

📁

Google Drive

Role: Document cloud storage.
Hosts non-code assets, specifically the scanned PDF containing all handwritten exam answers, generating a secure link.

🚀
🚀

Final Portal

Role: Deployment target.
Acts as the final aggregation point where all individual endpoints (GitHub, Drive, Colab URLs) are submitted for grading.
